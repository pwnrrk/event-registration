# Event Registration

This is assignment for Backend developer.

## Stack

- React.js (Vite + Typescript)
- Node.js (Express + Typescript)
- MongoDB

## Run The Application

Client

```bash
cd client
npm install
npm run dev
```

API

```bash
cd api
npm install
npm run serve
```

Initial Admin Password

- username: admin
- password: admin1234
